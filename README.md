# Sample Laboratory - Basic

기본 치과기공소 웹사이트 샘플 프로젝트

## 설명

이 프로젝트는 치과기공소를 위한 기본 웹사이트 템플릿입니다.

## 기술 스택

- HTML5
- CSS3
- JavaScript (Vanilla)

## 로컬 실행

```bash
python -m http.server 8000
```

또는

```bash
npx http-server -p 8000
```

브라우저에서 `http://localhost:8000` 접속

## 배포

정적 파일로 구성되어 있어 다음 플랫폼에 쉽게 배포 가능합니다:
- GitHub Pages
- Vercel
- Netlify
- Cloudflare Pages

## 라이선스

샘플 및 데모 목적으로 제작되었습니다.
